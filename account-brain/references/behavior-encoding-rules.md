# Behavior Encoding Rules

## Goal

Define how `account-brain` should convert creator assets into AI learning models.

## What This Skill Does

This skill turns captured creator assets into:

- repeatable rules
- structured models
- reusable generation priors

It is not a prose reporting skill.

## Encoding Principles

### 1. Extract behavior, not praise

Avoid statements like:

- "the account is strong"
- "the visuals are nice"
- "the copy is engaging"

Prefer:

- what pattern repeats
- what structural unit recurs
- what promise style is used
- what layout grammar is stable

### 2. Separate observed from inferred

Use three internal confidence modes:

- `observed`: directly supported by captured material
- `inferred`: likely, based on repeated signals
- `uncertain`: plausible but not well-supported

Do not elevate uncertain guesses into rules.

### 3. Model systems, not isolated artifacts

The skill should model:

- title system
- copy system
- visual system
- conversion system
- monetization system

Not just one post.

### 4. Encode what downstream generation needs

Prioritize fields that matter to:

- topic generation
- title generation
- copy generation
- image prompt generation
- page layout generation

## Things To Capture As Rules

### Positioning

- role identity
- target audience
- complexity level
- trust signal
- tone

### Titles

- formula patterns
- promise style
- urgency
- simplicity
- directness
- time compression

### Copy

- how the opening works
- whether the content is list-based / teaching-based / story-based / chart-based
- how dense the steps are
- how CTA appears

### Visuals

- whether the content is poster-like, chart-like, screenshot-like, note-like, or illustration-like
- color palette
- border system
- decorative grammar
- layout blocks
- visual density

### Funnel

- follow trigger
- collect trigger
- comment trigger
- homepage/DM trigger

### Monetization

- what appears to be sold
- whether monetization is direct or indirect
- whether the content acts as acquisition, trust building, or conversion
